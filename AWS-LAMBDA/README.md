# Project 4 - The Serverless Logic

## Objective
Build a Cost Calculator API using AWS Lambda.

## Services Used
- AWS Lambda
- IAM
- Amazon CloudWatch

## Language
Python

## Functionality
Accept two numbers as input and return their sum.

## Sample Input
{
  "num1": 15,
  "num2": 25
}

## Output
{
  "Sum": 40
}
