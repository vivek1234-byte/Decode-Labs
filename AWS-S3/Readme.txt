# Cloud Computing Project 1 – The Global Launch 🌍

## Project Overview
This project was completed as part of the DecodeLabs Industrial Training Kit (Batch 2026). The objective was to host a static portfolio website globally without using any servers by leveraging AWS S3 Static Website Hosting.

## Objective
- Create a static portfolio/resume website using HTML and CSS.
- Host the website on AWS S3.
- Enable public access and static website hosting.
- Generate a live, shareable URL accessible over the internet.
- http://vivek-portfolio-2006.s3-website-us-east-1.amazonaws.com

## Technologies Used
- HTML5
- CSS3
- AWS S3
- Static Website Hosting
- Git & GitHub

## Project Structure

```
├── index.html

```

## AWS Services Used
- Amazon S3 Bucket
- Static Website Hosting
- Public Access Configuration
- Bucket Website Endpoint

## Features
- Serverless static website hosting
- Publicly accessible website
- Low-cost cloud deployment
- Global availability
- Custom 404 Error Page

## Deployment Steps
1. Created an S3 bucket with a unique name.
2. Uploaded `index.html` and `404.html`.
3. Enabled Static Website Hosting.
4. Configured public access settings.
5. Accessed the website using the S3 Website Endpoint.
